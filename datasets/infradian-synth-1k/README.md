# INFRADIAN-SYNTH-1K

A CC-BY synthetic longitudinal cohort of menstrual cycles with coupled consumer-wearable signals, so
**anyone can run the entire [INFRADIAN benchmark](https://github.com/infradian/infradian) with no
data-use agreement.**

## What it is
Phenomenologically-generated daily records: literature-shaped hormone curves (LH surge at ovulation,
PdG luteal plateau, E3G periovulatory peak) coupled to wearable channels (relative skin temperature,
resting HR, RMSSD, respiratory rate, sleep) via **verified** published effect-size ranges. Includes
per-participant random effects, burst missingness, nuisance drivers, anovulatory cycles concentrated in
an irregular stratum, and a hydration nuisance term that makes the dilution negative control meaningful.

Three physiological facts are built in on purpose: temperature/HR are *lagged* progesterone proxies
(so prospective ovulation forecasting is ~0 skill by construction); an unmodeled-hydration path exists
for the T1-NC control; anovulatory cycles carry no luteal signature (basis of the anovulation task).

## Columns
`participant_id, segment_id, day_in_study, skin_temp_dev_c, rhr_bpm, hrv_rmssd_ms, resp_rate,
sleep_eff, steps, menses_reported, cramps, mood, stress, e3g, pdg, lh, phase, split, regularity`.

## NOT
Synthetic. Not real patient data. Calibrated to match published aggregate effect sizes, not derived
from a biophysical model. See `docs/effect_sizes.md` in the repo for every calibration source.
